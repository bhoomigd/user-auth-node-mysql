# user-auth-node-mysql# User Authentication API (Node.js + MySQL + JWT)

## 🚀 Project Overview

This is a REST API built using **Node.js**, **Express**, and **MySQL** with **JWT Authentication**.  
It provides secure user authentication and full CRUD operations.

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcryptjs
- dotenv
